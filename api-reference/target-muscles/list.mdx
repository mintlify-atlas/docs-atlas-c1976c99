**`GET /api/v1/targetMuscles`**

Returns all target muscle groups available in the exercise library. This endpoint takes no parameters.

<Tip>
Use the `targetMuscle` values returned here as the `targetMuscle` query parameter when calling `GET /api/v1/exercises` to filter exercises by target muscle.
</Tip>

## Known target muscles

There are 19 target muscle groups:

`abs` · `glutes` · `quads` · `hamstrings` · `pecs` · `lats` · `triceps` · `biceps` · `delts` · `calves` · `traps` · `spine` · `serratus anterior` · `levator scapulae` · `adductors` · `abductors` · `cardiovascular system` · `brachialis` · `brachioradialis`

## Response schema

<ResponseField name="totalTargetMuscles" type="number">
  Total number of target muscle groups (always `19`).
</ResponseField>

<ResponseField name="data" type="object[]">
  Array of target muscle objects.

  <Expandable title="Target muscle object properties" defaultOpen>
    <ResponseField name="targetMuscle" type="string">
      Target muscle name. Use this value as the `targetMuscle` filter on `GET /api/v1/exercises`.
    </ResponseField>
    <ResponseField name="imageUrl" type="string">
      Relative URL of the target muscle's category image (`.webp` format).
    </ResponseField>
    <ResponseField name="exerciseCount" type="number">
      Number of exercises in the library targeting this muscle.
    </ResponseField>
  </Expandable>
</ResponseField>
