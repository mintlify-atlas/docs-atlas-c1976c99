---
title: 'Get exercise'
description: 'Retrieve a single exercise by its unique ID.'
---

**`GET /api/v1/exercises/:id`**

Returns a single exercise object identified by its ID.

## Path parameters

<ParamField path="id" type="string" required>
  The unique exercise identifier. Exercise IDs are zero-padded strings (e.g. `"0001"`, `"0042"`).
</ParamField>

## Response schema

<ResponseField name="data" type="object">
  The exercise object.

  <Expandable title="Exercise object properties" defaultOpen>
    <ResponseField name="id" type="string">
      Unique exercise identifier.
    </ResponseField>
    <ResponseField name="name" type="string">
      Lowercase exercise name.
    </ResponseField>
    <ResponseField name="title" type="string">
      Display-formatted exercise title.
    </ResponseField>
    <ResponseField name="target" type="string">
      Primary target muscle group.
    </ResponseField>
    <ResponseField name="muscles worked" type="string">
      Markdown-formatted list of muscles worked.
    </ResponseField>
    <ResponseField name="bodyPart" type="string">
      Body part category for the exercise.
    </ResponseField>
    <ResponseField name="equipment" type="string">
      Equipment required to perform the exercise.
    </ResponseField>
    <ResponseField name="blog" type="string">
      Markdown-formatted long-form description, instructions, and tips.
    </ResponseField>
    <ResponseField name="images" type="string[]">
      Array of image URLs showing the exercise. URLs are relative (no `https://` prefix).
    </ResponseField>
    <ResponseField name="gifUrl" type="string">
      URL of an animated GIF demonstrating the exercise. Relative URL.
    </ResponseField>
    <ResponseField name="videos" type="string[]">
      Array of YouTube video URLs related to the exercise.
    </ResponseField>
    <ResponseField name="keywords" type="string[]">
      Search keywords associated with the exercise.
    </ResponseField>
  </Expandable>
</ResponseField>

## Examples

<CodeGroup>

```bash cURL
curl "https://api.bodyworks.akshatjaiswal.me/api/v1/exercises/0001"
```

```javascript JavaScript
const id = '0001';
const response = await fetch(
  `https://api.bodyworks.akshatjaiswal.me/api/v1/exercises/${id}`
);
const data = await response.json();
console.log(data);
```

</CodeGroup>

## Not found

If no exercise exists for the given ID, the API returns a `404`:

```json
{
  "error": "Endpoint not found",
  "message": "The requested resource does not exist"
}
```
