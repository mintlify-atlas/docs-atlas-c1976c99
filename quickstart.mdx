The Body Works API requires no authentication. All endpoints are public — you can start making requests immediately.

**Base URL:** `https://api.bodyworks.akshatjaiswal.me/api/v1`

<Steps>

<Step title="Fetch a list of exercises">

Make a GET request to `/exercises` to retrieve the first page of exercises.

<CodeGroup>

```bash curl
curl "https://api.bodyworks.akshatjaiswal.me/api/v1/exercises?page=1&limit=5"
```

```javascript JavaScript
const response = await fetch(
  "https://api.bodyworks.akshatjaiswal.me/api/v1/exercises?page=1&limit=5"
);
const data = await response.json();
console.log(data);
```

</CodeGroup>

You'll receive a paginated response:

```json
{
  "totalExercises": 1500,
  "totalPages": 300,
  "data": [
    {
      "name": "3/4 sit-up",
      "title": "3/4 Sit-Up",
      "target": "abs",
      "muscles worked": "- Rectus abdominis\n- Hip flexors\n- Lower back muscles\n",
      "bodyPart": "waist",
      "equipment": "body weight",
      "id": "0001",
      "blog": "...",
      "images": ["api.bodyworks.akshatjaiswal.me/assets/images/0001/1.jpeg"],
      "gifUrl": "api.bodyworks.akshatjaiswal.me/assets/gifs/0001.gif",
      "videos": ["https://www.youtube.com/watch?v=-B9lsTWsJCo"],
      "keywords": ["3/4 sit-up", "Abdominal exercise"]
    }
  ]
}
```

<Tip>
  Use the `totalPages` field to paginate through all results. Increment `page` from 1 to `totalPages`.
</Tip>

</Step>

<Step title="Filter exercises">

Narrow results using query parameters. Filter by body part, target muscle, equipment, or search text.

<CodeGroup>

```bash curl
# Exercises for the waist using body weight
curl "https://api.bodyworks.akshatjaiswal.me/api/v1/exercises?bodypart=waist&equipment=body%20weight&limit=10"
```

```javascript JavaScript
const params = new URLSearchParams({
  bodypart: "waist",
  equipment: "body weight",
  limit: "10",
});

const response = await fetch(
  `https://api.bodyworks.akshatjaiswal.me/api/v1/exercises?${params}`
);
const data = await response.json();
```

</CodeGroup>

You can combine multiple filters in a single request:

```bash
# Search for push exercises targeting the chest
curl "https://api.bodyworks.akshatjaiswal.me/api/v1/exercises?bodypart=chest&search=push&limit=10"
```

<Note>
  Use the [Body Parts](/api-reference/body-parts/list), [Target Muscles](/api-reference/target-muscles/list), and [Equipment](/api-reference/equipments/list) endpoints to get exact filter values.
</Note>

</Step>

<Step title="Get a single exercise by ID">

Each exercise has a unique string ID (e.g. `"0001"`). Fetch full details with:

<CodeGroup>

```bash curl
curl "https://api.bodyworks.akshatjaiswal.me/api/v1/exercises/0001"
```

```javascript JavaScript
const response = await fetch(
  "https://api.bodyworks.akshatjaiswal.me/api/v1/exercises/0001"
);
const { data } = await response.json();
console.log(data.title);       // "3/4 Sit-Up"
console.log(data.blog);        // Full instructional markdown
console.log(data.gifUrl);      // Animated demonstration
```

</CodeGroup>

</Step>

<Step title="Browse workout routines">

Use the `/routines` endpoint to fetch structured workout programs. Filter by goal, level, equipment, and more.

<CodeGroup>

```bash curl
# Beginner full body routines
curl "https://api.bodyworks.akshatjaiswal.me/api/v1/routines?level=beginner&type=full%20body&limit=5"
```

```javascript JavaScript
const params = new URLSearchParams({
  level: "beginner",
  type: "full body",
  limit: "5",
});

const response = await fetch(
  `https://api.bodyworks.akshatjaiswal.me/api/v1/routines?${params}`
);
const data = await response.json();

data.finalData.forEach((item) => {
  console.log(item.routine.routine_title);
  console.log(item.routine.workout_summary);
});
```

</CodeGroup>

<Warning>
  The routines endpoint is limited to **20 requests per 15-minute window**. Cache responses where possible. See [rate limits](/guides/rate-limits) for details.
</Warning>

</Step>

</Steps>
